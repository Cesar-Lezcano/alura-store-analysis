# Alura Store Analysis

## Descripción
Proyecto de análisis de datos enfocado en evaluar el rendimiento de tiendas minoristas utilizando Python, Pandas y Matplotlib. El objetivo es analizar ingresos, calificaciones, ventas por categoría y otros indicadores para identificar oportunidades de mejora y tomar decisiones basadas en datos.

## Objetivo
Determinar el rendimiento de cada tienda y proporcionar una recomendación basada en métricas cuantitativas como ingresos, satisfacción del cliente, productos más vendidos y costos de envío.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib

## Estructura del proyecto
El análisis se realiza en un notebook de Colab y los resultados se documentan con gráficos y conclusiones basadas en datos.

## Conclusión
Este proyecto demuestra habilidades en análisis de datos, visualización y toma de decisiones basadas en evidencia, aplicadas al contexto de retail.
